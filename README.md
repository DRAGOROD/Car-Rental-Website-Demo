Here's a modern **Car Rental Website** built using React. This project allows users to explore different car models, view pricing details, read customer reviews, and navigate smoothly across sections. The goal was to create a visually appealing and user-friendly interface similar to real-world rental platforms.

**⭐Project Link:-** https://DRAGOROD.github.io/Car-Rental-Website-Prototype/

## 🛠️ Tools & Technologies Used
- React
- Vite 
- JavaScript 
- HTML & CSS
- Git & GitHub 
- GitHub Pages 

## ✨ Features
- Smooth scrolling Animation
- Smooth navigation between sections
- Dynamic car model display with details
- Interactive FAQ section 
- Customer review section
- Responsive design for different screen sizes (Phone only currently)

## ⚠️ Problems Faced
- Deployment issues with GitHub Pages
- Images loading slowly after deployment
- Layout breaking on smaller screens

## ✅ Solutions Implemented
- Configured Vite base path correctly for GitHub Pages deployment
- Optimized images and added lazy loading to improve performance
- Added media queries and flexible layouts for responsiveness
